p Back To Top

This tag makes a new paragraph.

<p> Text </p>

ul Back To Top

Stands for "Unordered List". It creates a bulleted list by default.

  • List Item One
  • List Item Two

<ul> List </ul>

ol Back To Top

Stands for "Ordered List". It creates a numbered list by default.

  1. List Item One
  2. List Item Two

<ol> List </ol>

li Back To Top

This makes a list item that goes inside the "ol" or "ul" tags. Repeat the tag to add items to the list.

<li> List Item One </li>

a Back To Top

This tag creates a clickable link.

  • "href" Specifies the URL (web address) for a link

<a href= "google.com"> What you want the link to say. </a>

img Back To Top

This tag inserts an image from a link.

  • "src" Specifies the URL (web address) for an image
  • "alt" Specifies an alternative text for an image, when the image cannot be displayed

<img src= "guineaPig.jpg" alt= "Image Description">

html Back To Top

This is put right after the <!DOCTYPE HTML > tag and at the very end of a document so it's readable as an html file.

<html> HTML Code </html>

head Back To Top

The most common use of this tag is to contain the title of the document.

<head>

  <title> Document Title </title>  

</head>

link Back To Top

This links the document to an external source/stylesheet.

  • "rel" Specifies the relationship between the current document and the linked document
  • "href" Specifies the URL (web address) for a link

<link rel="stylesheet" href="example.css">

title Back To Top

This is where you put the title of the document.

<title> Document Title </title>

body Back To Top

This tag defined the document's body.

<body> Document Content </body>

span Back To Top

This tag is used to color text, for example:

She has green eyes.

  • "style" Specifies an inline CSS style for an element

<span style="color:green"> Text you want colored. </span>

div Back To Top

This tag divides/defines a section, for example if you wanted a specific line of text highlighted:

Lorem ipsum dolor sit amet consectetur adipisicing elit.

<div style="background-color:yellow">

 <p>Text you want highlighted. </p> 

</div>

header Back To Top

This header is where the introductory content for your document/webpage goes, such as a logo, icon, or headline.

<header> Headline or logo code. </header>

nav Back To Top

This tag is for computer navigation links to specified folders.

<nav>

  <a href="/html/"> HTML <nav>  

</nav>

footer Back To Top

This tag puts text or images at the footer of the document/webpage.

<footer> Text. </footer>

main Back To Top

This tag tag specifies the main content of a document/webpage, it can not be repeated.

<main> Main content code. </main>

section Back To Top

Similair to "div", this tag defines sections in a document, such as: chapters, headers, footers, etc.

<section> Header/footer code. </section>

aside Back To Top

This tag tag defines some content aside from the content it is placed in.

I visited the Cayman Islands.

(The Cayman Islands are south of Cuba.)

<h3> Main text. </h3>

<aside> Aside text. </aside>
